Official SDKs

Integrate VettedMe in minutes with our official SDKs

🐍

Python SDK

For Python 3.8+

pip install vettedme

Quick Start:

import vettedme

client = vettedme.Client(api_key="vm_live_...")
result = client.verify("PASS-ABC-123")

if result.valid:
    print(f"✅ {result.full_name} - {result.trust_score}% trust")

JavaScript/TypeScript SDK

For Node.js and browsers

npm install @vettedme/sdk

Quick Start:

import { VettedMeClient } from '@vettedme/sdk';

const client = new VettedMeClient({ apiKey: 'vm_live_...' });
const result = await client.verify('PASS-ABC-123');

if (result.valid) {
    console.log(`✅ ${result.fullName} - ${result.trustScore}% trust`);
}

CLI Tool

Command-line interface for developers

pip install vettedme-cli

Quick Start:

# Initialize
vettedme init

# Verify a passport
vettedme verify PASS-ABC-123

# List passports
vettedme passports list

# Manage webhooks
vettedme webhooks create --url https://example.com/webhook
