Getting Started

1. Get an API Key

Create an API key from your Dashboard

Authorization: Bearer vm_live_sk_...

2. Make Your First Request

Base URL: https://api.vettedme.ai

curl https://api.vettedme.ai/v1/verify \
  -H "Authorization: Bearer vm_live_sk_..." \
  -d "passport_id=PASS-ABC-123"

🔐 Passport Endpoints

POST /v1/passports

Issue a new digital passport with verified credentials

Request Body 
{
  "user_id": "user_123",
  "full_name": "Jane Doe",
  "email": "jane@example.com",
  "phone": "+1234567890",
  "date_of_birth": "1990-01-01",
  "license_number": "MD12345"
}
GET /v1/passports/{passport_id}/verify

Verify the authenticity of a passport

Response 
{
  "valid": true,
  "passport_id": "PASS-2026-JD-8B4C",
  "full_name": "Jane Doe",
  "trust_score": 99,
  "verified_at": "2026-07-14T18:00:00Z",
  "badges": ["medical_license", "education"]
}
GET /v1/passports/{passport_id}

Retrieve full passport details

GET /v1/passports

List all passports with pagination

🏅 Badge Endpoints

POST /v1/passports/{passport_id}/badges

Issue a new credential badge to a passport

DELETE /v1/badges/{badge_id}

Revoke a credential badge

🔔 Webhook Endpoints

POST /v1/webhooks

Create a new webhook endpoint

GET /v1/webhooks

List all webhook endpoints
