Frequently Asked Questions

Q: How do I get started with VettedMe?

Start by creating an account and generating an API key from the Dashboard. Then install one of our SDKs and follow the quick start guide.

Q: What is a VettedMe Passport?

A VettedMe Passport is a cryptographically signed digital credential that contains verified identity information. It uses Ed25519 signatures to ensure tamper-proof authenticity and can be instantly verified by anyone.

Q: How much does VettedMe cost?

VettedMe charges $0.10 per verification - 100x cheaper than traditional background checks. No monthly fees, no setup costs. Pay only for what you use.

Q: Is VettedMe HIPAA compliant?

Yes! VettedMe is SOC 2 Type II certified, HIPAA compliant, and ISO 27001 certified. View our Security & Compliance documentation for details.

Q: What programming languages do you support?

We currently support Python, JavaScript/TypeScript, and have a CLI tool. Ruby and Go SDKs are coming soon. Check out our SDKs page.

Q: How fast are API responses?

VettedMe API responses average under 50ms with 99.99% uptime. Verifications complete in less than 1 second.
